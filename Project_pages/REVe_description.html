REVe (The Re-imagined Environment Visualization engine)


Description

This is a framework built using Python and Pygame-CE mostly, other significant modules
incule modernGL and Python's socket, threading, math, etc. modules

brief history of the framework. REVe was at first built to decrease the amount of boilerplate code
I had to write if ever I needed to build something. I then later started adding more and more stuff
to it and then it became what it is today.

So what can it do? Well it can:

  • It allows one to create Simple or even complex GUI applications.
  • Thanks to the REVe.MGL submodule it allows you to create post-processing effects with the GPU
  • It has full compatabilitie with Pygame as it's a direct extention of Pygame-CE
  • Since it's built atop of Pygame-CE, it has support for Sound, User input and more.
  • It has a bunch of functions as well as built in functions to aid in development
  • And a bunch more stuff ...
